An Argument Against the Possibility of Being a Brain in a Vat:

Presupposition: A statement is true if and only if the terms in a statement refer to the realities they designate.  A statement is false or meaningless otherwise. 

For example, ‘Grass is green’ is true if and only if grass is green. 

1) Either we are brains in a vat or we are not brains in a vat.
Either -B or B   (assumption)

2) If  we are not brains in a vat, then we know that the sentence ‘We are brains in a vat’ is false.
If-B, then F  (assumption)

3) If we are brains in a vat, then we know that the sentence ‘We are brains in a vat’ is false.
If B, then F  (assumption)

4) So we know that the sentence ‘We are brains in a vat’ is false.
F         (from 1,2,3)

5) If we know that the sentence ‘We are brains in a vat’ is false, then we know that we are not brains in vats.
If F, then K   (from presupposition)

6) So we know that we are not brains in vats.
K    (from 4,5)
