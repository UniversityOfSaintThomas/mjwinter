Bonjour 1.5: Argument against Kant’s Rationalism:

 

Main Argument

1) If Kant is to be classified as a rationalist, then synthetic a priori (sap) knowledge must be justifiable by rationalist standards.

2) But Kant cannot provide a justification of sap by rationalist standards.

3) So, Kant should not be classified as a rationalist.

 

 

Argument for premise 2:

 

a) If sap is justifiable by rationalist standards, then it must be grounded in facts about reality. (p.15-16 BJ)

b) Kant’s justification of sap is not grounded in facts about reality.

(2) Kant cannot provide a justification of sap by rationalist standards.

 

 

Argument for (b):

 

If any sap proposition P is justified, then it will be justified by some proposition P*.

If P* is justified, then it must be known a posteriori or a priori.

If P* is known a posteriori, then P will be known a posteriori.

If P is known a posteriori, then P cannot be sap.

If P* is known a priori, then is must be analytic or synthetic.

If P* is synthetic, then there is a sap regress.

If P* is analytic, then P is true by definition.

So, either P is true by definition or sap regress.

If P is true by definition or there is a sap regress, then Kant’s justification of sap is not grounded in facts about reality.

b) Kant’s justification of sap is not grounded in facts about reality.
