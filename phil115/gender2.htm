Two Arguments Concerning Gender and the Human Person


An argument about gender roles that uses a premise from dualism

(1) There should be gender-specific roles only if men and women differ from one another significantly.

(2) But men and women do not differ from one another significantly.

(3) So, there should be no gender specific roles.
 
 
 

An argument about gender roles that uses a premise from hylomorphism

(1a) If there are significant differences between women and men, then there should be gender specific roles.

(2a) There are significant differences between men and women.

(3a) So, there should be gender specific roles.
