Complex Argument Exercises

Directions: Symbolize the following arguments and fill in the missing conclusion if possible.  If a conclusion is provided, mention whether the argument is valid.  If a fallacy is committed, explain what it is.

1) If the mind is not simply a brain process, then there is good reason to think that the soul exists.  And there is good reason to think that the soul exists.

2) If materialism is true, then hylomorphism is surely false.  But hylomorphism does not appear to be false.

3) If it rains then the ground is wet.  And the ground is wet.  Therefore it must have rained.

4) Humans deserve to be treated as ends in themselves if they have a rational nature.  And it is fairly obvious that we do have such a nature.

5) Unless I am severely mistaken, I was promised that my taxes would not be raised.  Campaign footage reveals that I am not mistaken.

6) Students will be awarded good grades only if they work hard.  Some students will be awarded good grades.

7) If you are comfortable with these exercises, then you will probably be prepared for the quiz.  It seems as though you are prepared for the quiz.

8) Unless everyone is ready, nobody goes.  Somebody goes.

9) George goes to the game unless Mary does.  May does not go.

10) George goes to the game only if Mary does.  Mary goes.

11) If Mary goes, Sam will not.  Sam goes.

12) Only if Sam goes, Jones will go.  Jones goes.
