     Phil 214: Introductory Ethics

      Lying

      Is it always wrong?

      • Immanuel Kant: "Yes"  Why?  Because it is irrational, it conflcits with the categorical imperative.
      • St. Thomas Aquinas: "Yes"   Why?  Because it conclicts with a basic human inclination.


      What about the hard cases?  In particular, what if a Nazi is pounding on the door asking if there are Jews in the house?  It would seem to be permissible to lie in this case to save a life.  But both Aquinas and Kant would say that we can never lie.  How can this claim be pausible in view of this tough case?

      Lying= making an assertion contrary to what is one one's mind.

      The wartime conditions make assertion impossible in the Nazi situation (Like a play or ballgame), so it would not be lying to say "There are no Jews here".  Why?  Because it would not be an assertion.
